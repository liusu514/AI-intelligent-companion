# 🤖 AI智能伴侣

一个基于 Streamlit 的智能对话助手，为你提供温暖贴心的陪伴体验。

## ✨ 功能特色

### 💬 智能对话
- 基于大语言模型的自然对话
- 流式输出，实时显示AI回复
- 记忆上下文，支持连续对话

### 🎭 个性化定制
- **伴侣名字**：给你的AI伴侣起一个专属名字
- **自定义性格**：描述你想要的伴侣性格，如温柔体贴、活泼开朗、高冷傲娇等
- 后台智能提示词自动适配用户设定的性格

### 💾 数据持久化
- 用户设置自动保存到 `ai history/user_settings.json`
- 聊天记录自动保存到 `ai history/chat_history.json`
- 刷新浏览器后数据不丢失

### 📋 多会话管理
- 支持新建多个独立对话
- 最多保留3个历史会话
- 一键切换历史会话
- 历史会话显示最后消息摘要和时间戳
- 支持删除不需要的会话

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Streamlit

### 安装依赖
```bash
pip install streamlit openai
```

### 配置API密钥
设置环境变量 `MIMO_API_KEY`：
```bash
export MIMO_API_KEY="你的API密钥"
```

### 运行应用
```bash
streamlit run "ai chatbot.py"
```

## 🎯 使用说明

1. **启动应用**：运行命令后，浏览器会自动打开应用页面
2. **设置伴侣**：在左侧边栏设置伴侣名字和性格
3. **开始对话**：在底部输入框输入消息，按回车发送
4. **管理会话**：
   - 点击「新建对话」创建新的会话
   - 点击历史会话切换到之前的对话
   - 点击 🗑️ 按钮删除会话

## 💡 小贴士

- 自定义性格越详细，AI的回复越符合你的期望
- 每次对话都会自动保存，不用担心数据丢失
- 历史会话带有时间戳，方便区分不同时段的对话

## 📝 技术栈

- **前端框架**：Streamlit
- **AI模型**：MIMO (via OpenAI API)
- **数据存储**：JSON文件
